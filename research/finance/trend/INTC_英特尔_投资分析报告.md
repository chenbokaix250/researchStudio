# Intel (INTC) 投资分析报告

> **巴菲特的裁决**：英特尔正处于艰难的转型期——从领先的芯片设计公司转向代工业务模式。这是一场豪赌：押注18A工艺节点能否吸引外部客户，同时AI加速器业务已被大幅收缩（Falcon Shores取消）。转型风险极高，业绩波动剧烈，不符合巴菲特"稳定可预测"的投资标准。当前估值看似便宜，但可能只是"价值陷阱"。

## 模块3：商业模式深度解析

### 一句话描述这个生意

> **英特尔设计芯片、制造芯片，并试图成为其他公司的芯片代工厂。**

### 收入模型

| 业务线 | 占比 | 状态 |
|--------|------|------|
| **客户端计算(CC)** | ~40% | PC芯片，持续萎缩 |
| **数据中心(DCAI)** | ~25% | 服务器CPU，被AMD/ARM侵蚀 |
| **网络与边缘(NEX)** | ~15% | 网络芯片，相对稳定 |
| **代工服务(IFS)** | ~5% | 新业务，增长中但规模小 |
| **Mobileye** | ~15% | 自动驾驶，独立运营 |

### 关键问题

- **制程落后**：7nm落后TSMC 2-3年
- **AI转型失败**：Gaudi市场份额极低，Falcon Shores已取消
- **代工客户稀缺**：AWS/Microsoft等客户订单规模未达预期

## 模块5：财务深度分析

### 关键财务指标（2024年估算）

| 指标 | 数值 | 评价 |
|------|------|------|
| **收入** | ~$54B | 同比下降 |
| **毛利率** | ~40% | 持续下降 |
| **净利率** | 负值 | 亏损状态 |
| **ROE** | 负值 | 资本回报差 |
| **自由现金流** | ~$2-3B | 勉强正数 |
| **净债务** | 高 | 资本开支巨大 |

### 所有者盈余计算

```
所有者盈余 = 净利润（亏损）
  + 折旧与摊销（~$6B）
  - 维护性资本开支（~$10B）
  ≈ -$4B至-$6B（负值）
```

**结论**：当前所有者盈余为负，业务处于消耗现金状态。

> 巴菲特不会投资这家公司：转型风险太高，护城河侵蚀，管理层持股低，业绩不可预测。

| 情景 | 目标价 | 概率 |
|------|--------|------|
| **乐观** | $30-35 | 20% |
| **基准** | $18-22 | 50% |

- **风险投资者**：小仓位博弈转型成功（<3%仓位）
- **价值投资者**：回避，这是价值陷阱而非价值投资
- **长期持有者**：等待转型成功证据后再考虑

1. 18A工艺量产进展
